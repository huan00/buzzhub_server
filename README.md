# Buzzhub

## Full stack social media app

### Features:

- User Auth
- create post with image, gif and video
- like post
- comment on post
- add/remove friendlist
- current weather display base on location

### Technology list:

#### Front end:

- React
- Redux
- MUI
- DropZone
- Formik
- Weather API

#### Backend Server:

- Django
- Rest_framework
- PostgresSQl

#### Coming soon features:

- Edit networking tabs
